作業重點
1. 利用lab2-2d-texture範例，設計一個2D game。 

2. 必需利用glut I/O的功能，觸發控制。 

3. 必需讀入至少三張不同於範例的圖片做為角色、敵方 
使得玩家可透過鍵盤或滑鼠的輸入，控制角色(2D圖片)的移動，完成某件具遊戲性的任務。 
(Ex: 在小精靈中，按下左鍵，會移動小精靈，如果吃到物品會加分，如果被敵人吃到會扣一條命)
